About PureCloud Apps

PureCloud Apps

PureCloud apps are a way to embed other applications to enhance the PureCloud Suite of tools.

Apps can be used to put a critical application in the hands of a contact center agent or to add entire vertical markets to the PureCloud platform.

Enabling

Apps are currently in development. For early access, please contact the PureCloud development team.

APIs

By utilizing PureCloud Authentication, backend SDKs, and javascript APIs, developers can build engaging apps that are tightly integrated with PureCloud.


API Desc
PureCloud App/UI SDK Use this api to interact with the PureCloud UI. Available on various CDNs and package managers.
PureCloud JS SDK A javascript implementation of the PureCloud SDK including authentication APIs. Also available in other languages for server-side purposes.
PureCloud Developer Center Lots of great information and documentation on developing for the PureCloud Platform.
