About Genesys Cloud Client Apps

Genesys Cloud Client Apps

Genesys Cloud Client Apps are a way to embed other applications to enhance the Genesys Cloud Suite of tools.

Apps can be used to put a critical application in the hands of a contact center agent or to add entire vertical markets to the Genesys Cloud platform.

APIs

By utilizing Genesys Cloud Authentication, backend SDKs, and javascript APIs, developers can build engaging apps that are tightly integrated with Genesys Cloud.


API Desc
Client App SDK Use this api to interact with the Genesys Cloud UI. Available on various CDNs and package managers.
Genesys Cloud JS SDK A javascript implementation of the Genesys Cloud SDK including authentication APIs. (Also available in other languages.)
Genesys Cloud Developer Center Lots of great information and documentation for developing on the Genesys Cloud Platform.
