About PureCloud Client Apps

PureCloud Client Apps

PureCloud Client Apps are a way to embed other applications to enhance the PureCloud Suite of tools.

Apps can be used to put a critical application in the hands of a contact center agent or to add entire vertical markets to the PureCloud platform.

APIs

By utilizing PureCloud Authentication, backend SDKs, and javascript APIs, developers can build engaging apps that are tightly integrated with PureCloud.


API Desc
Client App SDK Use this api to interact with the PureCloud UI. Available on various CDNs and package managers.
PureCloud JS SDK A javascript implementation of the PureCloud SDK including authentication APIs. (Also available in other languages.)
PureCloud Developer Center Lots of great information and documentation for developing on the PureCloud Platform.
